# Hán Note UI Starter

Starter UI for the AI Chinese learning system.

Includes:

- `web-next`: React/Next.js web app UI
- `mobile-expo`: React Native/Expo mobile app UI

The UI is built around the product flow:

1. Analyze Chinese text
2. Select token status
3. Read Vietnamese explanations
4. Save vocabulary
5. Review due words
6. Manage vocabulary and notification settings

Both apps use mock data and local state so the UI can be wired to real API endpoints later.
